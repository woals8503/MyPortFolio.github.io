기술분야 상세내용
컬렉션 List<> 인터페이스
List<>와 ArrayList<> 클래스를 활용하여 데이터를 DB에 DML 할 때 사용했습니다.
JSTL JSP 페이지 내에서 스크립틀릿을 사용하지 않고 JSTL을 사용하여 자바 코드를 바로 사용하지 않고 로직을 내장하였습니다.
MVC 패턴 JSP 파일 내에서 자바 코드를 사용하지 않고 모델, 뷰, 컨트롤러로 각각 페이지를 나누어 코드를 작성함으로써 가독성을 높이고, 오류 발생 시 오류 수정에 시간을 아낄 수 있도록 했습니다
JavaScript JSP에서 조건 체크 시 JavajScript를 사용하였습니다.
Mybatis 오라클DTO, MAP등으로 쉽게 매핑하여 코드량을 줄였습니다.
jQuery JavaScript의 라이브러리인 jQuery를 활용하여 코드를 간략히 하였습니다.
BootStrap 부트스트랩을 활용함으로서 기능에 더 집중하였습니다.

편스토랑

편스토랑은 바쁜 현대 사회에서 어디에나 존재하는 편의점의 음식들을 활용해 요리를 잘 못해도 할 수 있는 쉬운 요리 레시피를 공유하는 사이트 입니다. 평소 편의점 식품으로 혼밥을 자주하는데 나만의 레시피를 공유하고 다른 이에게도 공유하면 좋겠다싶어 개발하였습니다.

  • Category : 요리 레시피
  • Project date : 2022.9.15 ~ 2022.9.28
  • Git URL : github.com/2206PublicDataWebApp/pyeonstaurant

구현 기능

  • 공지사항 게시판 : 기본적인 CRUD방식으로 구현하였습니다. 관리자로 로그인 시 boolean값으로 체크하여 등록 수정 삭제가 가능합니다.

  • 관리자 페이지 : 레시피 조회,삭제 기능 / 회원 관리(조회, 강제탈퇴) / 게시물, 댓글 신고 조회 및 관리 / 포인트 관리
