List<>와 ArrayList<> 클래스를 활용하여 데이터를 DB에 DML 할 때 사용했습니다.
JSTL JSP 페이지 내에서 스크립틀릿을 사용하지 않고 JSTL을 사용하여 자바 코드를 바로 사용하지 않고 로직을 내장하였습니다.
MVC 패턴 JSP 파일 내에서 자바 코드를 사용하지 않고 모델, 뷰, 컨트롤러로 각각 페이지를 나누어 코드를 작성함으로써 가독성을 높이고, 오류 발생 시 오류 수정에 시간을 아낄 수 있도록 했습니다
JavaScript JSP에서 조건 체크 시 JavajScript를 사용하였습니다.
Ajax JSP에서 조건 체크 시 Ajax를 사용하여 컨트롤러로 데이터를 주고받기 위해 사용하였습니다.
Mybatis 오라클DTO, MAP등으로 쉽게 매핑하여 코드량을 줄였습니다.
jQuery JavaScript의 라이브러리인 jQuery를 활용하여 코드를 간략히 하였습니다.
BootStrap 부트스트랩을 활용함으로서 기능에 더 집중하였습니다.
Web Socket 웹소켓을 이용하여 서버와 클라이언트의 통신을 주고받았습니다.
결제 API 결제 API를 통해 예약시스템을 구현하였습니다.
휴대폰 인증 API 휴대폰 API를 통해 본인인증 서비스를 구현하였습니다.

제주어때

제주도 여행객을 위한 숙박시설 정보제공과 근처 제주여행객과의 실시간 채팅을 나눌 수 있는 웹사이트입니다. 여행과 새로운 만남을 좋아하는 저와 같은 사람들에게 즐거운 제주여행이 되지않을까 하는 마음에 개발했습니다.

  • Category : 숙박시설 및 커뮤니티
  • Project date : 2022.10.04 ~ 2022.11.16
  • Git URL : https://github.com/2206PublicDataWebApp/Jeju

구현 기능

  • 숙소 목록 : 숙소 리스트 필터링 중복적용(가격, 서비스시설, 날짜 등 변경값 유지) 가능하도록 구현하였습니다.

  • 예약 : 휴대폰인증, 결제를 통한 예약시스템입니다. 회원은 30분 결제 대기 서비스를 이용할 수 있으며, 30분간 카운트다운 타이머가 이용내역에 보여집니다(페이지를 벗어나도 시간 유지 localstorage Session 활용). 시간내에 미결제시 자동삭제됩니다.

  • 실시간 알림 : 결제대기 상태에서 남은시간 10분일때 사용자에게 결제 남은시간을 알려줍니다.(Web Socket과 localstorage Session 응용)

  • 쿠폰 : 쿠폰을 1회발급받을 수 있습니다. 이용완료시 쿠폰은 더이상 사용할 수 없으며 예약중 또는 결제대기 상태에서 예약취소시 다시 사용할 수 있습니다.
